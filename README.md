# 🚀 AI Resume Analyzer SaaS

🔍 AI-powered Resume Screening Tool with Login System, Dashboard Analytics & NLP-based Resume Matching.

---

## 🌟 Features

- 🔐 User Authentication (Login / Signup)
- 📄 Resume Upload (PDF)
- 📊 Match Score (%) based on Job Description
- 🤖 AI Feedback (Missing Skills + Suggestions)
- 📈 Dashboard with Score Analytics
- 👥 Multi-user Support
- ⚡ Fast & Lightweight (No API required)

---

## 🛠️ Tech Stack

- 🐍 Python
- 📊 Scikit-learn (NLP)
- 🎨 Streamlit (UI)
- 📄 PyPDF2 (PDF parsing)
- 📉 Matplotlib (Charts)
- 🗄️ SQLite (Database)
